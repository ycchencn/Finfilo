高性能多市场股票数据 RESTful API

覆盖 A股/ST/ETF、美股、港股行情查询、历史K线计算与实时 Tick 数据管理。内置 Redis 缓存层与向量化数据管道，专为量化研究与金融数据分析设计。

🚀 快速开始 📖 查看 API 文档

✨ 核心特性

架构简洁 · 性能优先 · 开箱即用

🌍

多市场覆盖

统一路由抽象，支持 CN/US/HK 三大市场数据接入，标准化响应结构。

⚡

高性能缓存

Flask-Caching + query_string 细粒度控制，有效削峰填谷，降低数据库压力。

🐼

向量化管道

Pandas 批量操作替代逐行 apply，K线解析性能提升 300%+，内存占用更低。

🔒

严格参数校验

内置股票代码正则校验器，非法请求直接拦截，保障接口安全与数据一致性。

📦

标准响应体

全局封装 json_resp，统一 code/data/msg 结构，前后端对接零摩擦。

💾

Tick 数据流

支持批量上传与按标的过滤读取，自动剥离非核心字段，适配高频辅助场景。

📖 核心接口预览

所有接口均返回 JSON，成功状态码 `code: 0`。完整路由见源码 `blueprints/`。

路径方法说明关键参数
/cn/stocksGETA股全量列表-
/cn/st_stocksGETST板块列表-
/cn/stock/infoGET个股详细信息symbol
/cn/stock/profileGET公司基本面资料symbol
/us/hk/{market}/historyGET历史K线行情symbol, start_date, end_date
/cn/tick/upload_tick_allPOST批量上传最新 TickJSON Body
/cn/stock/tickGET获取指定标的 Ticksymbol

🚀 快速开始

  1. 克隆仓库git clone https://github.com/your-repo/global-stock-api.git && cd global-stock-api
  2. 环境准备python -m venv .venv && source .venv/bin/activate
  3. 安装依赖pip install -r requirements.txt
  4. 配置 Redis 缓存设置环境变量 export REDIS_URL=redis://localhost:6379/0
  5. 启动服务flask run --host=0.0.0.0 --port=8080
